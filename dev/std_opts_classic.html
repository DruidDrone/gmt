Common Options¶

STANDARDIZED COMMAND LINE OPTIONS

-Binformation

Specify map frame and axes parameters (…)

-Jparameters

Select map projection (…)

-K

Append more PS later (…)

-O

This is an overlay plot (…)

-P

Select Portrait orientation (…)

-Rwest/east/south/north[/zmin/zmax][+r]

Specify Region of interest (…)

-U[label][+c][+jjust][+odx/dy]

Plot time-stamp on plot (…)

-V[verbosity]

Run in verbose mode (…)

-X[a|c|r]off[u]

Shift plot origin in x-direction (…)

-Y[a|c|r]off[u]

Shift plot origin in y-direction (…)

-aname=col,…

Associates aspatial data with columns (…)

-bi[ncols][type][w][+l|b]

Select binary input (…)

-bo[ncols][type][w][+l|b]

Select binary output (…)

-d[i|o]nodata

Replace columns with nodata with NaN (…)

-e[~]”pattern” | -e[~]/regexp/[i]

Filter data records that match the given pattern (…)

-f[i|o]colinfo

Set formatting of ASCII input or output (…)

-g[+]x|X|y|Y|d|Dgap[u][+n|p]

Segment data by detecting gaps (…)

-h[i|o][n_headers]

ASCII [I|O] tables have header record[s] (…)

-icolumns

Selection of input columns (…)

-je|f|g

Mode of spherical distance calculation (…)

-n[type][+a][+bBC] [+c][+tthreshold]

Set grid interpolation mode (…)

-ocolumns

Selection of output columns (…)

-pazim[elev[/zlevel]][+wlon0/lat0[/z0]][+vx0/y0]

Control 3-D perspective view (…)

-r[g|p]

Sets grid registration (…)

-s[cols][+a|+r]

Control treatment of NaN records (…)

-ttransparency

Set layer transparency (…)

-x[[-]n]

Set number of cores in multi-threaded modules (…)

-:[i|o]

Expect y/x input rather than x/y (…)
